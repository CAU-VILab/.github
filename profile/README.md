# Visual Intelligence Lab (VILab)
### Chung-Ang University, South Korea  

---

The goal of the VILab is to develop useful technologies on visual intelligence which can be applied to visual surveillance systems, deepfake detection, intelligent factory, contents, food analysis, and etc. We are targeting on the advancement of the visual intelligence including the detection, tracking, and recognition of human/vehicles, and other objects. In addition, we also developing AI technologies such as probabilistic machine learning models and deep learning algorithms.
